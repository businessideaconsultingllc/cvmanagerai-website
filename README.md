# CV Manager AI Website

Official website for CV Manager AI - Create professional CVs with AI technology.

## Website Structure

```
website/
├── index.html          # Landing page
├── privacy.html        # Privacy policy
├── terms.html          # Terms of service
├── support.html        # Support center
├── CNAME              # Custom domain configuration
├── css/
│   └── styles.css     # Main stylesheet
├── js/
│   └── main.js        # JavaScript functionality
└── app/               # Flutter web app
```

## Features

- **Modern Landing Page**: Beautiful, responsive design with gradient backgrounds
- **Legal Pages**: Complete privacy policy and terms of service
- **Support Center**: FAQ and contact information
- **Flutter Web App**: Full CV Manager AI application at `/app/`

## Deployment

The website is deployed on GitHub Pages with custom domain:
- **URL**: https://cvmanagerai.com
- **App**: https://cvmanagerai.com/app/

### Deploy Updates

1. Make changes to website files
2. Commit and push to GitHub:
   ```bash
   git add .
   git commit -m "Update website"
   git push origin main
   ```

## Technologies

- HTML5
